# Security Policy

## Supported Versions
We support the following versions of OffroadTerrain AI with security updates:
| Version | Supported |
| --- | --- |
| 1.0.x | ✅ |
| < 1.0.0 | ❌ |

## Reporting a Vulnerability
If you discover a security vulnerability within this project, please send an e-mail to security@offroad-ai.example.com. All security vulnerabilities will be promptly addressed.
